	외부에 공통값을 설정해두면 내부 어플리케이션에서 그 값을 참조하여 사용!
	별도의 환경설정 파일을 생성하여 관리

리스너 -> 초기 데이터 세팅
	자바객체를 공유하기위한 수단
	컨테이너(서버,톰캣)가 수행하는 이벤트를  모니터링하다가 특정 상황에서 액션을 수행하는 
	특수한 형태의 서블릿
서블릿
	ex) 초기데이터, 샘플데이터




필터 -> 국제화(UTF-8)
	브라우저(클라이언트화면, 사용자) -- 요청 --> C -- M에서 받은 데이터 --> V	
	ex) 국제화, 인증확인, 로깅(모니터링), 데이터 가공
	어플리케이션을 지원하기 위한(사용자 요청에 대한 다양한 기능을 제공하기 위한) 특수한 형태의 서블릿	
	사용자의 요청 처리 이전에 메서드를 먼저 수행함
	사용자 요청에따른 서블릿이나 JSP가 실행되기전에 res, req객체를 탈취하여 객체를 변경할수있음!!





	MVC★
	M: DB연동 / JDBC / VO,DAO
	V : 사용자에게 제공되는 페이지 / .jsp
	C : 페이지 <-> 페이지 / 사용자의 요청에 대한 데이터 확보
